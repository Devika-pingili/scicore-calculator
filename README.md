# Scientific Calculator Web Application

## Overview

This project is a modern **Scientific Calculator Web Application** built using **HTML, CSS, and JavaScript**.
It supports basic arithmetic operations, advanced mathematical functions, trigonometric calculations, and complex expressions using parentheses.

The calculator is designed with a **modern dark-mode interface**, responsive layout, and smooth button interactions. It also supports **keyboard input and calculation history**, making it suitable for both quick calculations and advanced mathematical operations.

This project is part of my developer portfolio and demonstrates my skills in **front-end development, JavaScript logic implementation, and UI design**.

------------------------------------------------------------------------------------------------

# Features

## 1. Basic Operations

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Percentage (%)
* Decimal number support

------------------------------------------------------------------------------------------------

## 2. Mathematical Functions

The calculator supports several advanced mathematical functions:

* Square root (√)
* Cube root (∛)
* Square (x²)
* Cube (x³)
* Power (xʸ)
* Absolute value |x|
* Factorial (!)
* Logarithm (log)
* Natural logarithm (ln)

------------------------------------------------------------------------------------------------

## 3. Trigonometric Functions

Trigonometric calculations are performed in **degrees**:

* sin(x)
* cos(x)
* tan(x)

Examples:

* sin(30)
* cos(60)
* tan(45)
* sin(30) + cos(60)

------------------------------------------------------------------------------------------------

## 4. Mathematical Constants

The calculator includes commonly used constants:

* π (pi)
* e (Euler’s number)

Examples:
π * 2
e²

------------------------------------------------------------------------------------------------

## 5. Expression Support

The calculator can evaluate complex mathematical expressions using parentheses.

Examples:
(5 + 3) * 2
√(16)
∛(27)
2^3 + (5 * 4)

------------------------------------------------------------------------------------------------

## 6. User Interface

* Modern **dark-mode scientific calculator design**
* Large display screen
* Responsive layout for different screen sizes
* Grid-based button layout
* Smooth button hover and click animations

------------------------------------------------------------------------------------------------

## 7. Additional Features

* Calculation history
* Keyboard input support
* Clear (C) button
* Delete (⌫) button
* Error handling for invalid operations (e.g., division by zero)
